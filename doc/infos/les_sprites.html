Les fichiers de données

Les sprites (extensions .icn)

Contenu des fichiers

Les fichiers icn se décomposent en répertoires contenant des fichiers images (png car la transparence du fond est souvent requise) et un fichier spec.xml contenant le nb de fichiers, puis pour chaque fichier (numéroté de 1 à n) le nom du fichier image à utiliser et des offsets de placement en nb de pixels.

Liste des principaux fichiers

La modification du theme passe par le remplacement des fichiers suivants :

  • cbkg*.icn

    13 fichiers représentant les types de terrain utilisés pour les combats. Il n'y a qu'une image par fichier représentant un paysage plat et de taille 640x443. les 99 pixels du haut de l'image représentent l'horizon qui est inaccessible aux combattants. Le reste de l'image représente le champ de bataille sur lequel seront placés des obstacles aléatoirement avant le début de chaque combat.

  • minimon.icn

    594 images utilisées comme animations de créatures sur la carte d'aventure. Certaines de ces animations correspond au mouvement effectué par les créatures pour attaquer les héros.

    Il est envisageable d'utiliser les images du fichier mons32.icn mais dans ce cas on se prive des animations des monstres sur la carte d'aventure.

  • miniport.icn

    Les 60 portraits miniatures des héros en taille 30x22.

    Ils sont utilisés dans la partie droite de l'interface et dans les pop-up.

  • mons32.icn

    Contient des miniatures de tous les monstres semblables à celle qui aparaissent sur la carte d'aventure. La différence étant que les miniatures de la carte sont animées. Ces miniatures aparaissent dans les pop-up pour donner l'ordre d'attaque durant les combats mais surtout dans la partie droite de l'interface. Elles sont utilisées également dans l'éditeur de cartes. leur taille varie mais ne dépassent pas 41x39 (pour les dragons). Malgré ces variations de taille, les offsets sont quasiment toujours nuls, à part quelques valeurs à 1 pixel.

    020.png correspond aux fées

    027.png correspond aux licornes (32x37 de taille originale)

    029.png correspond aux centaures (33x29 de taille originale)

  • <monstres>.icn

    Un fichier/répertoire par créature (soit 66 fichiers). Contient les animations des différents monstres sur le champ de bataille.

    On retouve les animations suivante : attente, avance, attaque vers le haut, attaque au centre, attaque vers le bas, réception d'un coup, mort.

    Pour les créatures attaquant à distance, il faut ajouter les trois animations suivantes : tir vers le haut, tir au centre, tir vers le bas.

  • port00*.icn

    101x93

  • portmedi.icn

    51x47

  • townbkg[0-5].icn

    640x256

    Les images de fond des chateaux. Respectivement : chevalier, barbare, sorciere, warlord, magicien, et nécromancien.
