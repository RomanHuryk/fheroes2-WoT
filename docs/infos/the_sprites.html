The data files

The sprites (files with .icn extension)

files content

For each icn file, the extraction program will create a directory containing one or more images. As these images often require transparency, they will be saved as png files. A file named spec.xml will also be generated, containing the number of images, and for each of them (numbered from 1 to n), the corresponding filename and the offsets in pixels to draw the image. To precisions on how these informations are store in icn files, see the icn format description.

Main files list

To making a mod needs the modification of the following files :

  • cbkg*.icn

    13 files for the differents battlefield background. There is only one image by icn file, which is a landscape image of size 640x443. The 99 first lines of pixels are for the horizon (sky, far mountain) and are not part of the battlefield. Some obstacle will be randomly placed on the rest of the image before each battle begin.

  • minimon.icn

    594 images used for animate the monsters on the adventure map. Some of these animations are the movement made by the monster to attack a heroe's army.

    The data of file mons32.icn can be used instead of the one of minimon.icn but there will not be monster animations on the aventure map.

  • miniport.icn

    The 60 mini portraits of the heroes of size 30x22.

    They are used in the rigth part of the interface and in the pop-up.

  • mons32.icn

    The mini representations of all the monsters, like those on the adventure map, but without the animations. They appear on the pop-up windows, on top of the screen during the battle to give the turns order, and also on the rigth part of the interface. They are also used in the maps editor. Images' size varies but never exceeds 41x39, which is the dragons' miniatures sizes. In spite of the size variations, the offsets are almost always zero, except when their value is 1.

    020.png are for sprites (fairies)

    027.png are for licorns (original size of 32x37)

    029.png are for centaurs (original size of 33x29)

  • <monster>.icn

    One icn file (or directory) by monster (66 files). This file contains the animations of the monsters in the battlefield.

    The animations are : waiting, move forward, attack to the top, attack on front, attack to the bottom, receive an attack, die.

    For the monsters with remote attack, to the contact attack we have to add the three animations to the remote attacks : to the top, to the middle, and to the bottom.

  • port00*.icn

    One file by heroe, for the maximun size portrait (101x93). This portrait appears in the heroe's view window.

  • portmedi.icn

    Medium size portraits of the heroes, with size 51x47.

  • townbkg[0-5].icn

    Six images of size 640x256. They are the background images of the castles. Respectively : knight, barbarian, witch, warlord, wizard, and necromancer.
